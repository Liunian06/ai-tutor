
## 四轨学习法

四轨学习法是一套面向高等教育课程和技术学习的学习框架。

核心顺序是：

```text
先用人话听懂，再用教材学懂，再用题目练懂，最后用应用用懂。
```

它把一个知识点的学习拆成四个轨道：

| 轨道 | 目标 | 解决的问题 |
|---|---|---|
| 第 0 轨：通俗引入轨 | 建立直觉 | 先听懂人话版本 |
| 第 1 轨：课程理解轨 | 理解正式知识 | 看懂教材、定义、公式和原理 |
| 第 2 轨：考试得分轨 | 转化为答题能力 | 会做题、会答题、会复盘错题 |
| 第 3 轨：应用发现轨 | 连接真实场景 | 知道知识能在哪里用、怎么做小任务 |

完整方法文档见：

```text
docs/four_track_learning_method.md
```

## Learning Engine Skill

`learning-engine` 是一个 Claude Code Skill，用于把四轨学习法转化为可执行的学习流程。

它会围绕学习材料生成：

- 学习路线图；
- 分节 lesson 文件；
- 四轨结构化讲解；
- 苏格拉底式理解检查；
- Bloom 分层考试训练；
- 应用发现任务；
- 学习进度记录；
- 阶段报告或最终报告。

技能文件位置：

```text
.claude/skills/learning-engine/SKILL.md
```

## 方法整合

本项目不是简单叠加多个学习理论，而是以四轨学习法为主框架，重新安置常见教学方法：

| 方法 | 在本项目中的位置 |
|---|---|
| 四轨学习法 | 全局主框架 |
| 苏格拉底式追问 | 放入第 1 轨，用来检查课程理解 |
| Bloom 层级 | 放入第 2 轨，用来组织考试训练难度 |
| 逆向设计 | 放入第 3 轨，用小任务、小产出倒推应用路径 |

## 适用场景

适合用于：

- 大学课程学习；
- 计算机、数学、统计、工程类课程；
- 考试复习；
- 技术文档学习；
- 开源项目学习；
- 把课本知识迁移到真实应用。

## 使用方式

在 Claude Code 中安装或加载本仓库中的 skill 后，可以这样使用：

```text
使用 learning-engine 帮我学习线性代数第一章
```

或者：

```text
使用 learning-engine 阅读这个项目，并生成四轨学习路线图
```

对于小知识点，也可以直接使用紧凑四轨模式：

```text
请用四轨学习法讲解“数据库索引”
```
